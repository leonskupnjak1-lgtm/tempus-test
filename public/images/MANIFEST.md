# Image manifest

Every slot below is filled with a real photograph sourced from the
tempus-pool.hr production site, selected per `tempus-pool-asset-library.md`
(the asset-library research doc kept alongside this project). If a path is
ever emptied out, `ImagePlate` (`src/components/common/ImagePlate.jsx`) falls
back to a styled placeholder plate automatically — nothing else needs to
change in the code.

**Provenance note:** files named with a phone-camera timestamp (e.g.
`20220430_201033`) are near-certainly Tempus Pool's own photos. Files named
with a long numeric ID (e.g. `370477394`) follow the pattern of photos
sourced from a booking/listing platform for the referenced villa — authorship
should be confirmed with the client before this goes to production. See
Section 4 of the asset-library doc for the full caveat.
